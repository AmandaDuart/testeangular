Secretaria Online

O objetivo dessa aplicação é testar as habilidades técnicas dos(as) novos(as) instrutores de desenvolvimento web front-end.

A generation está buscando instrutores para facilitar treinamentos para jovens beneficiados pelo Programa Generation -Desenvolvedor Java Jr. O Programa busca qualificar e inserir jovens no mercado de trabalho com uma formação rápida no estilo bootcamp.

O instrutor será responsável por facilitar sessões nas quais os jovens desenvolverão habilidades técnicas e socioemocionais para qualifica-los para vagas em empregadores parceiros da Generation.
